# Pygame 3D Engine

## Create ENV

```
conda env create -n pg3d-env -f ./env.yml
```

## Update ENV

```
conda env update -n pg3d-env -f ./env.yml
```

## Remove ENV

```
conda env remove --n pg3d-env
```

## Activate ENV

```
conda activate pg3d-env
```

## RUN

```
python src/main.py
```
